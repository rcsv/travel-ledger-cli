# v4.7.37 — P-6l cross-day move planning (documentation-only)

## Summary

**v4.7.37** adds a **planning-only specification** for the next structural intent **`move_itinerary`**.
This intent covers moving **exactly one** itinerary item from a **source Day** to a **destination Day** within the **same Trip**, including insertion into the destination Day order.

Design boundary (preserved):

```text
reorder_itinerary = same-day sort_order change
move_itinerary    = cross-day day_id + sort_order change
```

Spec: [v4.7.37-p6l-cross-day-move-planning.md](../specifications/v4.7.37-p6l-cross-day-move-planning.md)
