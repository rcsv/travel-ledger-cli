# v4.7.10 — Proposal Envelope show / inspect

## Summary

**v4.7.10** implements **P-2**: `proposal show <file>` and `proposal inspect <file>` for human-readable review of Trip Proposal Envelope files — reusing v4.7.9 validation, without DB or materialize.

Spec: [v4.7.10-proposal-envelope-show-inspect.md](../specifications/v4.7.10-proposal-envelope-show-inspect.md)

## What changed

- **New:** `proposal show`, `proposal inspect` commands
- **Extended:** `ProposalEnvelopeView` — validation report + structural details
- **Not changed:** `trip validate-export`, schema v8, JSON golden

## Usage

```bash
caglla proposal show docs/public/examples-non-normative/trip-proposal-envelope.example.json
caglla proposal inspect envelope.json
```

## Compatibility

- **JSON golden:** unchanged
- **Markdown golden:** colophon version only (`4.7.10`)
