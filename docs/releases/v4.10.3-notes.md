# v4.10.3 — Desktop settings navigation polish

## Summary

**v4.10.3** adds a small Settings navigation shell to the read-only desktop app:

- Settings entry pinned at the bottom of the trip sidebar
- Settings view with **Database** and **About** sections
- Move Change / Forget Database out of the browsing header into Settings → Database
- Keep a clear **Open Database** action when no database is selected
- Preserve trip selection when opening and leaving Settings
- About shows the app version via Tauri `getVersion()` (no frontend hardcoding)
- Root Cargo version bumped to `4.10.3`; Markdown colophon synced

Implementation spec: [v4.10.3-desktop-settings-navigation-polish.md](../specifications/v4.10.3-desktop-settings-navigation-polish.md)

## Packaging note

- GitHub Release assets remain **CLI 3OS only**
- Desktop remains **developer preview / source-only / no bundle** (`bundle.active = false`)

## Boundary

- No Profile / user preference persistence (Profile remains a future candidate)
- No write UI / write facade
- No desktop bundle distribution
