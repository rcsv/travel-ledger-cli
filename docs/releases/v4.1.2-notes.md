# v4.1.2 — Okinawa Travel Book sample enrichment

## Summary

**v4.1.2** implements the [v4.1.1 enrichment plan](specifications/v4.1.1-okinawa-travel-book-sample-enrichment-plan.md): Travel Book fixture data in the Okinawa Sesoko 2026 canonical sample. Adds Trip/Day summaries, Trip/Day notes, reservations, and planned-cost estimates while preserving the post-trip ledger invariants (58 itineraries, 49 expenses, ¥561,780 Actual, 6 receipts).

Requires **v4.1.1** baseline. Trip export still writes **`schema_version: 8`**. `trip export-md` layout is unchanged — **v4.2.0**.

Implementation: [v4.1.2-okinawa-travel-book-sample-enrichment-implementation-plan.md](../specifications/v4.1.2-okinawa-travel-book-sample-enrichment-implementation-plan.md)
