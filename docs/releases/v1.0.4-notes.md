# v1.0.4 Checklist export/import and release binaries

## Summary

This release completes Checklist backup/restore in JSON import/export and publishes aligned multi-platform binaries via GitHub Actions. Version metadata (`Cargo.toml`, `caglla --version`, release assets) now matches `1.0.4`.

Note: [v1.0.3](https://github.com/rcsv/caglla-cli/releases/tag/v1.0.3) was a prerelease used to validate the Release workflow only.

## Changes

- Include Checklist in `trip export` / `trip import`:
  - Export JSON adds `checklist_items`
  - Import restores checklist titles, completion state, and sort order with new IDs
  - Legacy export files without `checklist_items` remain importable
- Add GitHub Actions Release workflow:
  - Builds Linux / macOS (Apple Silicon) / Windows release binaries on `v*` tag push
  - Uploads `.tar.gz` / `.zip` assets to GitHub Releases
- Update README install and release history for v1.0.4

## Not in this release

Future roadmap items remain out of scope (no implementation in v1.0.4):

- Participant management
- Expense tracking
- Multi-currency support
- Settlement / splitting
