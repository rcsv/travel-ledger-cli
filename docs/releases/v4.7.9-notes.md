# v4.7.9 — Proposal Envelope file validation

## Summary

**v4.7.9** implements **P-1** from the v4.7.8 plan: `proposal validate <file>` for Trip Proposal Envelope JSON — separate from `trip validate-export` (schema v8 Trip only).

Spec: [v4.7.9-proposal-envelope-file-validation.md](../specifications/v4.7.9-proposal-envelope-file-validation.md)

## What changed

- **New:** `proposal validate <file>` with blocking errors / non-blocking warnings
- **New:** `src/proposal/` module, test fixtures, CLI integration tests
- **Not changed:** schema v8 export contract, JSON golden structure, `trip validate-export` behavior

## Usage

```bash
caglla proposal validate docs/public/examples-non-normative/trip-proposal-envelope.example.json
caglla proposal validate envelope.json --json
```

## Compatibility

- **JSON golden:** unchanged
- **Markdown golden:** colophon version only (`4.7.9`)
