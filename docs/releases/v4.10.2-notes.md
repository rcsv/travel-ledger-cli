# v4.10.2 — Desktop read-only usability polish

## Summary

**v4.10.2** polishes the read-only desktop shell before write paths:

- Clearer Trip list selection, count, and metadata (destination / currency / dates when set)
- Trip detail layout with days count; omit empty metadata
- Day tabs and timeline show weekday/date; itinerary empty states
- Sequence-first activity timeline (not a task checklist; no itinerary completion)
- Softer developer-preview copy; Forget still never deletes the SQLite file
- Root Cargo version bumped to `4.10.2`; Markdown colophon synced

Implementation spec: [v4.10.2-desktop-read-only-usability-polish.md](../specifications/v4.10.2-desktop-read-only-usability-polish.md)
