# v1.0.0 First stable CLI baseline

## Summary

v1.0.0 marks the first stable baseline of Caglla Travel CLI, a local SQLite-based command-line tool for managing trips, itineraries, checklists, diagnostics, and exports.

## Highlights

- Trip CRUD
- Itinerary CRUD
- Checklist CRUD
- Timeline display
- Trip stats
- Trip doctor / advisor
- JSON export/import/diff
- Markdown export
- JSON output for read-oriented commands
- GitHub Actions CI
- Release notes and verification workflow

## Notes

- Data is stored locally in SQLite.
- Web and cloud sync are not included.
- JSON output is currently intended for tooling and automation and may evolve in future versions.

## Validation

- `cargo fmt`
- `cargo clippy -- -D warnings`
- `cargo test`
- `make check`
- GitHub Actions `Rust CI`
