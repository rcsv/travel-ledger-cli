# v2.0.0 Release Notes (draft)

## Participant Foundation

- **`participants` テーブル** — Trip スコープの参加行（`name`, `sort_order`, `is_self`）
- **CLI** — `participant add` / `list` / `show` / `update` / `delete`
- **Export schema v4** — top-level `participants[]`（`is_self` 含む）。v3 import 互換を維持
- **diff / export-md / stats / doctor** — Participant 差分・一覧・人数統計・info 警告

詳細: [participant-implementation-plan.md](../specifications/participant-implementation-plan.md)

- Person / Traveler Profile
- Expense `paid_by_participant_id`
- Settlement / Reservation guest linking
