## Purpose

```text
src/ 配下がフラットになりすぎて見通しが悪くなってきたため、
責務ごとに module を整理する。
```

v3.2.1 は **構成整理のみ**。CLI 挙動・JSON / DB schema・export/import 形式は変えない。

---

## Non-goals

```text
- 新機能追加
- CLI コマンド名・出力形式の変更
- JSON schema 変更
- DB schema / migration 仕様変更
- export/import 形式変更
- Repository pattern 導入
- models.rs の全面分割（移動のみ）
- Planned vs Actual difference
- DB パス設定（--db / CAGLLA_DB / db use）
- Trip / Itinerary 等の command handler 一括移動（将来）
```

## Target layout

```text
src/
  main.rs

  cli/           Clap struct / enum 定義
  commands/      CLI 固有の実行ハンドラ（about, db）
  domain/        Caglla 旅行ドメインモデル（models）
  storage/       SQLite 接続・migration・DB status
  analysis/      doctor / advisor / statistics
  io/            markdown / diff（外部表現・比較）
  output/        JSON 等の出力補助（print_json）

  （ルートに残置 — 段階移行）
  checklist.rs, day.rs, estimate.rs, expense.rs, itinerary.rs,
  money.rs, note.rs, participant.rs, reservation.rs, summary.rs, trip.rs
```

### 責務

| module | 責務 |
|---|---|
| **cli/** | Clap 定義 |
| **commands/** | CLI サブコマンドの実行ハンドラ |
| **domain/** | 旅行ドメイン構造体 |
| **storage/** | DB 接続・migration・DB status |
| **analysis/** | 旅行データの検査・助言・集計 |
| **io/** | Markdown export / diff 等 |
| **output/** | pretty JSON 等の共有出力 |
