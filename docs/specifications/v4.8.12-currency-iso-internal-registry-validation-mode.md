## 1. Implemented API / model

### 1.1 `CurrencyValidationMode`

```rust
pub(crate) enum CurrencyValidationMode {
    FormatOnly,  // default — legacy
    IsoStrict,   // registry + denylist
}
```

| Mode | 責務 |
|---|---|
| `FormatOnly` | trim → uppercase → ASCII alpha-3。未知コード **許可**（既存 default） |
| `IsoStrict` | format 後、denylist → ISO registry 照合。非 ISO / denylist **reject** |

### 1.2 Public internal API（`src/money/mod.rs`）

```rust
// 既存 — FormatOnly（互換 wrapper）
pub(crate) fn validate_currency_code(code: &str) -> Result<String>

// 新規 — mode 指定
pub(crate) fn validate_currency_code_with_mode(
    code: &str,
    mode: CurrencyValidationMode,
) -> Result<String>
```

`parse_amount_for_currency` / `format_amount_*` は **`validate_currency_code()`（FormatOnly）** 経由 — 変更なし。

### 1.3 Registry module（`src/money/currency_registry.rs`）

```rust
pub(crate) fn is_iso4217_alpha3(code: &str) -> bool
pub(crate) fn is_travel_expense_denylisted(code: &str) -> bool
```

## 2. ISO registry 方針

### 2.1 Scope

- **262 codes** — sorted static slice + `binary_search`
- **Primary source:** `iso_currency` 0.5.3 `isodata.tsv`（182 entries）— **参照元のみ、Cargo dependency は追加していない**
- **Supplemental:** withdrawn / historical codes not in crate dataset（例: `LTL`, `LVL`, `DEM`, `ITL`, `MRO`）
- **Active + withdrawn:** historical travel records 互換のため withdrawn を許容

### 2.2 必須 travel currency（確認済み）

`JPY`, `USD`, `EUR`, `KRW`, `GBP`, `TWD`, `CNY`, `HKD`, `AUD`, `CAD`, `CHF`, `SGD`, `THB`

### 2.3 非 ISO 例（strict reject）

`ZZZ`, `ABC`, `JPN`

---

## 3. Denylist 方針

**strict mode のみ。** format-only では既存互換（`XXX` 等も通る）。

| Code | 理由 |
|---|---|
| `XXX` | ISO "no currency" — 旅行費用に意味なし |
| `XTS` | testing code |
| `XAU`, `XAG`, `XPT`, `XPD` | precious metals |
| `XBA`, `XBB`, `XBC`, `XBD` | bond / composite units |

Error（strict）: `currency は旅行費用として許可されていません: {code}`

Registry unknown（strict）: `currency は有効な ISO 4217 コードではありません`

Format errors: 既存 message 維持
