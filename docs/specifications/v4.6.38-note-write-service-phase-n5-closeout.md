### Note 周辺 helper 一覧

| Helper | read write | adapter? | 判定 |

| `print_note_detail(&Note)` | show + add/update | **いいえ** — domain 直接、正式経路 | **維持** |

**判定:** production write path に legacy adapter は **存在しない**。N-5 実装 cleanup は **不要**。
