## 2. 責務境界

v4.1.2 追加データは、既存の責務分離を守る。

| 種別 | 責務 | Travel Book | Okinawa enrichment |
|---|---|---|---|
| **Summary** | Trip / Day の **要約（Abstract）** | Trip overview / Daily schedule 冒頭 | Trip 1 + Day 4 |
| **Note** | **補足・注意・運用メモ**（Annotation） | Notes 章 / Day 補足 | Trip 2 + Day 4 |
| **Reservation** | 実旅行で **予約・手配に相当** したもの | Reservations 章 | 5 件（下表） |
| **Estimate** | **Planned** 費用 | Planned cost 章 | 7〜9 件（fixture） |
| **Expense** | **Actual** 実績 | 旅行前しおりでは控えめ（v4.0.0） | **変更しない** |
| **Receipt** | 未整理支払い候補（Actual でも Planned でもない） | **表示しない** | **変更しない** |

### 2.1 Summary vs Note

```text
Summary = その Trip / Day をざっと理解する短文（導入文）
Note    = 運用上の注意・調整余地・同行者向け補足
```

Itinerary の `note` フィールド（単行 Remark）は既存どおり Itinerary 配下に残す。Note **entity** は Trip / Day スコープで追加する。

### 2.2 Reservation ルール

```text
- 実旅行で予約・手配に相当した Itinerary にのみ付与する
- 実績を「架空の事前予約」として偽装しない
- confirmation code は台帳・PDF に実在根拠がある場合のみ記載
- 不明な confirmation は作らない（フィールド省略または null）
- remark に fixture 由来を明記する（例: canonical sample fixture — derived from actual ledger）
```

### 2.3 Estimate ルール

```text
- Travel Book の Planned cost 章を検証するための planned fixture
- Expense / Actual から「当初こう予定していた」と主張しない
- 金額は代表値・丸め値でよい（Planned 表示のサンプル目的）
- note に fixture 由来を明記する（例: Travel Book planned-cost fixture; not original pre-trip budget）
- 実績と同額にする必要はない
```

### 2.4 Receipt

```text
- seed の Receipt Inbox（6 件 / pending ¥15,980）は変更しない
- Travel Book には出さない（v4.0.0 / v4.1.0 維持）
- README で「しおりに混ぜない」負の例として継続説明する
```

## 4. Reservation 対応表

**正本キー:** `Day` + `sort_order` + `Itinerary title`（seed 投入順が変わっても追跡可能）。

**想定 itinerary ID** は現行 `seed.sh`（v0）の投入順からの参考値。v4.1.2 で seed 末尾に追記する場合は ID がずれるため、実装時は **タイトル + day + sort_order** で再解決する。

| # | Day | sort_order | Itinerary title | 想定 ID | reservation_type | provider / 内容 | confirmation | remark / 期間 |
|---|---:|---:|---|---:|---|---|---|---|
| R1 | 1 | 3 | P1 G Parking | 3 | `parking` | セントレア P1 G Parking | `EXAMPLE-PARKING-001`（Expense note 根拠） | fixture 注記。出発日駐車。 |
| R2 | 1 | 6 | NU045 NGO ⇒ OKA (11:00着) | 6 | `flight` | 航空便 NU045（NGO⇒OKA） | **なし**（台帳に便名のみ） | fixture 注記。 |
| R3 | 4 | 17 | NU046 OKA ⇒ NGO (22:05着) | 54 | `flight` | 航空便 NU046（OKA⇒NGO） | **なし** | fixture 注記。 |
