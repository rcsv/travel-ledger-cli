## 1. Command 責務

| Command | 責務 | exit code |
|---|---|---|
| `proposal validate` | valid / invalid 判定 | invalid → 非 0 |
| `proposal show` | 概要表示 | Trip export 混同 / parse 失敗 → 非 0；validation invalid でも **0** |
| `proposal inspect` | 構造・validation 詳細 | 同上 |

---

## 2. `proposal show` 表示項目

Title, Destination, Date policy, Created at, Valid until, Expired, Source, Provider, Candidate days/itineraries count, Missing fields / Assumptions / Hint warnings count, Validation result, Blocking errors count, Validation warnings count, blocking errors 一覧（あれば）

---

## 3. `proposal inspect` 追加項目

Metadata, Date policy 詳細, Notes, Materialize hints（required decisions / missing_fields / assumptions / warnings）, Candidate days / itineraries preview, Validation warnings 詳細
