## Status

| 観点 | 判定 |
|---|---|
| `note_add` service | **追加** — `NoteAddParams` / `NoteAddServiceResult` |
| `note_update` service | **追加** — `NoteUpdateParams` / `NoteUpdateServiceResult` |
| `note_delete` service | **追加** — `NoteDeleteParams` / `NoteDeleteServiceResult` |
| add/update result | **`Note`** — read `note_show` と同型 |
| delete result | **`{ id, title }`** — enrich なし |
| handler | service 経由 + `print_note_detail` |
| `note.rs` mutation | **残置** — service から呼び出し |
| owner 解決 | **`note_add` service 内** — handler から除去 |
| enriched layer | **なし** |
| JSON output | **不変** |
| human output | **不変**（`note_cli` tests） |
| `import_export_notes` | **未変更** |

---

## Write flow（実装後）

```text
note add:
  handler → NoteAddParams
  service → resolve_note_owner_for_add → add_note → get_note
  handler → println! + print_note_detail

note update:
  handler → NoteUpdateParams
  service → update_note → get_note
  handler → println! + print_note_detail

note delete:
  handler → NoteDeleteParams
  service → get_note (snapshot) → delete_note
  handler → println! + title display
```
