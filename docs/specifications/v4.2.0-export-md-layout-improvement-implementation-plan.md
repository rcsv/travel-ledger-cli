# v4.2.0 — export-md layout improvement (implementation)

Implements [v4.1.0 Travel Book chapter structure design](v4.1.0-travel-book-chapter-structure-design.md) in `src/io/markdown.rs`.

| Item | Status |
|---|---|
| Release | v4.2.0 |
| Scope | `trip export-md` Markdown layout only |
| Validation fixture | [Okinawa Sesoko 2026 sample](../../samples/okinawa_sesoko_2026/README.md) |

## Implemented chapters

| Chapter | Heading | Notes |
|---|---|---|
| Cover | `# {trip}` + dates + traveler count | Trip `summary` not on cover |
| Trip overview | `## Trip overview` | Summary, participants table, operational stats |
| Daily schedule | `## Daily schedule` | `### Day N — date`, day summary, `####` itineraries |
| Reservations | `## Reservations` | After daily schedule |
| Checklist | `## Checklist` | Omitted when empty |
| Planned cost | `## Planned cost` | Estimates grouped by itinerary |
| Notes | `## Notes` | Note entity only (not itinerary `note` field) |
| Colophon | `## Colophon` | Generator credit, version, timestamp |

## Omitted from Travel Book output

Per v4.1.0 §7:

- Expense / Actual cost lines
- Planned vs Actual Difference
- Receipt / Pending Receipt
