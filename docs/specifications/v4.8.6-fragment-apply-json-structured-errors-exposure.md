## 1. JSON report 変更

### 1.1 Additive field

`FragmentApplyDryRunReport` に `structured_errors` を JSON 直列化対象とする。

```json
{
  "schema_version": 2,
  "errors": ["add_expense は itinerary target のみサポートしています（trip / day は未対応）"],
  "structured_errors": [
    {
      "code": "APPLY_UNSUPPORTED_TARGET",
      "kind": "blocking",
      "message": "add_expense は itinerary target のみサポートしています（trip / day は未対応）",
      "phase": "simulate",
      "intent": "add_expense",
      "target_type": "trip"
    }
  ],
  "warnings": [],
  "required_decisions": []
}
```

### 1.2 Empty array omission

success report（`structured_errors` 空）では、既存 optional field と同様 **`structured_errors` キー自体を省略**（`skip_serializing_if = "Vec::is_empty"`）。

### 1.3 schema_version

`FRAGMENT_APPLY_REPORT_SCHEMA_VERSION = 2` を維持。破壊的変更ではなく unknown-field 許容の additive 追加のため bump 不要。

## 2. structured_errors[] 要素形状

### 2.1 Required fields（v4.8.6）

| Field | 型 | 説明 |
|---|---|---|
| `code` | string | stable registry（`APPLY_*`） |
| `kind` | string | `blocking` / `warning` / `decision_required` |
| `message` | string | human-readable（unstable） |
| `phase` | string | `fragment_validate` / `gate` / `confirm_scope` / `simulate` / `confirm_transaction` |

### 2.2 Optional metadata（存在時のみ出力）

| Field | v4.8.6 |
|---|---|
| `field_path` | ✓ wired path で使用 |
| `intent` | ✓ |
| `target_type` | ✓ |
| `entity_type` / `entity_id` / `target_id` / `expected` / `actual` / `retryable` / `blocking` / `help` | 未使用 — 将来候補 |

## 3. 露出対象（v4.8.5 wiring 踏襲）

v4.8.5 limited wiring 済み structured errors が JSON に露出される。新規 wiring は行わない。

| Category | Codes（代表） |
|---|---|
| unsupported intent / target | `APPLY_UNSUPPORTED_INTENT`, `APPLY_UNSUPPORTED_TARGET` |
| target resolver | `APPLY_TARGET_UNRESOLVED`, `APPLY_TARGET_AMBIGUOUS`, `APPLY_ENTITY_NOT_FOUND`, `APPLY_FIELD_INVALID` |
| required decisions | `APPLY_REQUIRED_DECISION` |
| simulate blocking | `APPLY_MUTATION_FIELD_REJECTED`, `APPLY_ENTITY_PARENT_MISMATCH`, `APPLY_BASELINE_MISMATCH` |

**未露出（registry のみ / 未 wiring）:** `APPLY_PREVIEW_MISMATCH`, `APPLY_TOCTOU_DRIFT`, `APPLY_SCOPED_WRITE_*`, `APPLY_TRANSACTION_FAILED`
