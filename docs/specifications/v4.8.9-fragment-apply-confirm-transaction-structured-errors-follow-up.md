## 1. Scope（本 release）

### 1.1 confirm baseline mismatch

| 項目 | 値 |
|---|---|
| `code` | `APPLY_BASELINE_MISMATCH` |
| `phase` | `confirm_transaction` |
| 検出 | `revalidate_delete_estimate_before_write` 内 `detect_delete_estimate_baseline_conflicts` |
| message 例 | `baseline mismatch: expected_amount (99999) が現行値 (15000) と一致しません` |
| optional `field_path` | `candidate_content.{expected_key}`（存在時のみ） |

### 1.2 scoped DELETE 0 rows

| 項目 | 値 |
|---|---|
| `code` | `APPLY_SCOPED_WRITE_ZERO_ROWS` |
| `phase` | `confirm_transaction` |
| 検出 | `delete_estimate_row_scoped` が affected rows = 0 |
| message | 従来互換（`削除できませんでした — DB 更新しません`） |
| DB safety | transaction rollback / no partial write 維持 |

### 1.3 scoped DELETE multiple rows

| 項目 | 値 |
|---|---|
| `code` | `APPLY_SCOPED_WRITE_MULTIPLE_ROWS` |
| `phase` | `confirm_transaction` |
| 検出 | `delete_estimate_row_scoped` が affected rows > 1 |
| message | 従来互換（`exactly one row contract 違反`） |
| テスト | DB 制約上 integration 再現困難 — **classifier / helper-level test** |
