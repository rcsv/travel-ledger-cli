## 1. Overview

v3.2.1 は **behavior-preserving module layout refactor**。`src/` のフラット構成を責務別 module に整理し、以降の機能追加・保守の見通しを改善した。

### 到達点

| 項目 | 判定 |
|---|---|
| **新機能** | 追加なし | ✅ |
| **CLI 挙動** | コマンド名・引数・出力は変更なし | ✅ |
| **JSON schema** | 変更なし（`domain/models.rs` 定数・構造体維持） | ✅ |
| **DB schema / migration** | 変更なし（`storage/db.rs` ロジック維持） | ✅ |
| **export/import 形式** | 変更なし | ✅ |
| **`db path` / `db status`** | open 前 dispatch 維持 | ✅ |
| **`db reset`** | 従来どおり | ✅ |
| **`make check`** | PASS | ✅ |

**結論:** ユーザー向け挙動は v3.2.0 と同等。変更は `src/` 内部の module 配置と import パスのみ。

```text
- Trip / Itinerary 等 command handler の commands/ 一括移動

- models.rs の細分化

- DB パス切替（--db / CAGLLA_DB / db use）
```
