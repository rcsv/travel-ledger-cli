# v4.6.33 — Expense write service Phase W-2+W-3

## Status

| 観点 | 判定 |
|---|---|
| `expense_add` service | **追加** — `ExpenseAddParams` / `ExpenseAddServiceResult` |
| `expense_update` service | **追加** — `ExpenseUpdateParams` / `ExpenseUpdateServiceResult` |
| `expense_delete` service | **追加** — `ExpenseDeleteServiceResult` |
| add/update result | **`ExpenseEnrichedPart`** — read `expense_show` と同型 |
| delete result | **`{ id, amount, currency }`** — enrich なし |
| handler | service 経由 + `print_expense_detail_from_enriched` |
| `expense.rs` mutation | **残置** — service から呼び出し |
| `print_expense_detail` | **write から除去** — `#[allow(dead_code)]` adapter として残置 |
| JSON output | **不変** |
| human output | **不変**（`expense_cli` tests） |
| `import_expense_v3` | **未変更** |

---

## Write flow（実装後）

```text
expense add:
  handler → ExpenseAddParams
  service → parse_* → add_expense → get_expense → enrich_expense
  handler → println! + print_expense_detail_from_enriched

expense update:
  handler → ExpenseUpdateParams
  service → get_expense → parse_* → update_expense → enrich_expense
  handler → println! + print_expense_detail_from_enriched

expense delete:
  handler → id
  service → get_expense (snapshot) → delete_expense
  handler → println! + amount display
```

## Service API

### `expense_add`

```rust
pub struct ExpenseAddServiceResult {
    pub id: i64,
    pub expense: ExpenseEnrichedPart,
}
pub fn add_expense(conn: &Connection, params: ExpenseAddParams) -> Result<ExpenseAddServiceResult>
```

### `expense_update`

```rust
pub struct ExpenseUpdateServiceResult {
    pub expense: ExpenseEnrichedPart,
}
pub fn update_expense(conn: &Connection, params: ExpenseUpdateParams) -> Result<ExpenseUpdateServiceResult>
```

### `expense_delete`

```rust
pub struct ExpenseDeleteServiceResult {
    pub id: i64,
```
