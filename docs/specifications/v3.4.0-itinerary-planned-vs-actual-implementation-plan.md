## Difference vs Balance / Settlement

**Difference（本 release）** は予定と実績の金額差のみを表す。

```text
Difference = Actual − Planned
```

**Balance / Settlement（対象外）** は誰が払い・誰が負担し・誰が誰に事前入金したか・最終的に誰が誰へいくら払うべきかを扱う **別レイヤー**。

| 観点 | Difference | Balance / Settlement |
|---|---|---|
| 式 | `Actual − Planned` | payer / beneficiary / transfer / net balance |
| 事前入金・取りまとめ | **表現しない** | 将来 Participant / Settlement 系列 |
| v3.4.0 | **Itinerary 表示のみ** | **非対象** |

旅行取りまとめ者への事前入金がある場合、最終的な支払いバランスは Difference では表現できない。Settlement は v4.x 以降または Participant 系列の大きな設計テーマとして defer する。

## Definitions

| 概念 | 定義（Itinerary 単位） | 正本 |
|---|---|---|
| **Planned** | 当該 Itinerary 配下 Estimate の通貨別合計 | Estimate 行の集計 |
| **Actual** | 当該 Itinerary 配下 Expense の通貨別合計 | Expense 行の集計 |
| **Difference** | `Actual − Planned`（通貨別、derived） | **v3.4.0 で新規** |

### Difference の式

v3.3.0 の Trip 単位と **同一**:

```text
difference_totals[currency] =
  expense_totals.get(currency).unwrap_or(0)
  - estimate_totals.get(currency).unwrap_or(0)
```

**keys** = `estimate_totals` の key ∪ `expense_totals` の key（`BTreeMap` 昇順）。

| 符号（Actual − Planned） | 意味 |
|---|---|
| **negative** | 予定より少なく使った |
| **positive** | 予定より多く使った |
| **zero** | 一致 |

### 通貨ルール

- 通貨別に差分（**為替換算なし**）
- 片方にしかない通貨は欠損側 **0**

### 表示 gate（Itinerary 単位）

```text
itinerary_estimate_count > 0 && itinerary_expense_count > 0
```

| 状況 | 予定費用表 / Expenses | Planned / Actual / Difference サマリー |
|---|---|---|
| Estimate のみ | 予定費用表のみ | **Difference 省略** |
| Expense のみ | Expenses のみ | **Difference 省略** |
| 両方あり | 既存明細 + **サマリー追加** | **表示** |
| どちらもなし | 省略 | **省略** |

Trip 単位 gate（v3.3.0）とは **独立**。Itinerary A は両方あり・Itinerary B は Estimate のみ、という混在を許容する。

追加候補（実装時に選択）:

| 方針 | 内容 |
|---|---|
| **A（推奨）** | `sum_amounts_by_currency(estimates)` / `sum_amounts_by_currency(expenses)` を statistics.rs に追加し、Itinerary スライスから `BTreeMap` を構築 → `compute_difference_totals` を呼ぶ |

**`analysis/difference.rs` は v3.4.0 でも作らない**。

### `format_itinerary_item_markdown` 拡張

[src/io/markdown.rs](../../src/io/markdown.rs) — 既存の「予定費用」表・Expenses 明細の **後** に、gate 通過時のみサマリーを追加:

```markdown
Planned total:
  - JPY 14,000
Actual total:
  - JPY 13,750
Difference:
  - JPY -250
```

| 項目 | 仕様 |
|---|---|
| ラベル | **`Planned total:` / `Actual total:` / `Difference:`**（英語 — v3.3.0 / Overview と統一） |
| 金額 | `crate::money::format_amount_value` |
| 通貨順 | `BTreeMap` 昇順 |
| 既存明細 | 予定費用表・Expenses 行は **変更しない**（サマリーは追加のみ） |
